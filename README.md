# Kullanıcı Yönetim Uygulaması

Kullanıcıları, gönderileri, albümleri ve yapılacakları yönetmek için bir React uygulaması. React, TypeScript ve Vite ile oluşturulmuştur.

![Uygulama Ekran Görüntüsü](./src/assets/screenshot.png)

## Özellikler

- **Kullanıcı Listesi**: Tüm kullanıcıların listesini görüntüleyin.
- **Kullanıcı Detayları**: Her kullanıcı hakkında detaylı bilgileri inceleyin.
- **Gönderiler**: Kullanıcı gönderilerini okuyun ve etkileşimde bulunun.
- **Albümler**: Kullanıcı fotoğraf albümlerine göz atın.
- **Yapılacaklar**: Kullanıcı görevlerini takip edin.
- **Favoriler**: Favori içeriklerinizi kaydedin.

## Teknoloji Yığını

- **Frontend**: React, TypeScript
- **Derleme Aracı**: Vite
- **Stillendirme**: Tailwind CSS
- **Durum Yönetimi**: Zustand
- **Yönlendirme**: React Router

## Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone <repository-url>
   ```

2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

3. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```

## Kullanım

- Kullanıcı listesini görmek için ana sayfaya gidin.
- Detaylarını, gönderilerini, albümlerini ve yapılacaklarını görmek için bir kullanıcıya tıklayın.
- İçerikleri yer imlerine eklemek için favoriler özelliğini kullanın.
